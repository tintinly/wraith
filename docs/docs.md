# 项目身份

## 项目名称

my-personal-page·

## 项目描述

这是一个个人页面项目，用于展示个人信息、项目经历、技能等。

# 技术栈与工具

## 前端

- 框架：Vue.js 3
- 构建：Vite 4
- 样式：Tailwind CSS 3
- 路由：Vue Router 4
- 逻辑/类型检查：TypeScript
- 图标：iconify + unplugin-icons (旧方案)
- 图标：iconify + tailwindcss-icons

## 基础设施

- 域名：tintinly.top

## 开发工具

- 包管理器：npm 11
- 代码编辑器：VS Code
- 版本控制：Git
- 代码托管平台：GitHub

## 命名规范
- 变量/函数：camelCase（`getUserInfo`, `isLoading`）
- 组件：PascalCase（`UserProfile`, `TimeTracker`）
- 常量：UPPER_SNAKE_CASE（`MAX_RETRY_COUNT`, `API_BASE_URL`）
- 类型/接口：PascalCase（`User`, `ApiResponse<T>`）
- 私有方法：下划线前缀（`_validateInput`, `_cacheResult`）
